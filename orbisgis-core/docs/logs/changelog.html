Version 3.0b (2010-07-21)

New features

  • New plugin system
  • Add translations FR and EN
  • Add online help menu
  • Add workspace window chooser
  • Add timer to save workspace
  • Add WMS crs filter
  • Add functions filter
  • Add where filter to select row in a table
  • Improve database plugin to display schema and view
  • Export plugin in Geocatalog
  • Add pick coordinate tool
  • Add basic rendering options
  • Add new legends (proportional lines)
  • New SQL and beanshell console document syntax
  • New grammar syntax : drop table if exists, drop table purge, delete, alter add column
  • New grammar functions : st_planarGraph, st_graph, ST_Normalize, ST_IsRectangle, ST_Snap, ST_PointsToLine, ST_D8Accumulation, ST_D8AllOutlets, ST_D8ConstrainedAccumulation, ST_D8Direction, ST_D8RiverDistance, ST_D8Slope, ST_D8StrahlerStreamOrder, ST_D8Watershed, ST_FillSinks, ST_LSFactor, ST_StreamPowerIndex, ST_WetnessIndex, ST_InteriorPoint, ST_SetZ, ST_RasterAlgebra, ST_Count, ST_Shadow, ST_ToMultiSegments, ST_RemoveRepeatedPoints, ST_AddPoint, ST_AddPoints,ST_Densify, ST_Scale, ST_Rotate, ST_BoundingCircle, ST_MinimumRectangle, ST_OctogonalEnvelope, ST_NearestPoints, ST_CircleCompacity
